# Overlook Hotel

## About
The goal of this project was to create a very basic hotel management app that could allow guests as well as managers to log in and view/edit room reservations. 

## Set-Up

1. clone down this repo to your machine
1. navigate to the cloned repo and run `npm install`.
1. in your terminal, run `npm run start`
1. in your browser, type `localhost:8080` (or whatever server your terminal says the local server is running on)

login: to login, the username has to be `manager` or `customerXX` where `XX` is any number between 0-50 (if choosing a number less than 10, be sure to add a zero before the digit. eg: customer04 or customer29). The password is `overlook2019`.
